# Ayush Mishra — Data Analytics Portfolio

Simple, responsive portfolio site to showcase projects, certificates, and contact details. Built with HTML, CSS, and JavaScript and ready to deploy on GitHub Pages.

## Preview Locally
1. Open `index.html` in your browser, or
2. Start a local server:
   - Python: `python3 -m http.server 8000` then visit `http://localhost:8000/`

## Deploy on GitHub Pages
1. Push the site to your GitHub repository.
2. In repository Settings → Pages:
   - Source: Deploy from a branch
   - Branch: `main` and folder `/root`
3. Wait for Pages to build, then visit `https://<your-username>.github.io/Portfolio-Website-/`

## Content
- Summary, Experience, Projects, Education, Involvement, Skills, Certificates, Contact
- Images in `assets/` and `images/`; icons rendered with white backgrounds for dark theme visibility

## Customize
- Update text in `index.html`
- Replace or add assets in `assets/` or `images/`
- Adjust theme colors in `styles.css` under `:root`
